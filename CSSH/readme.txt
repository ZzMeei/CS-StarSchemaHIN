baseline : NaiveOnline(BasicQuerySlow.java) and FastOnline(BasicQueryFast.java).
bean : Two object classes, MetaPath and MetaPathSet.
csh : codes reused from CSH ("Effective and Efficient Community Search over Large Heterogeneous Information Networks").
index : index related codes, KC, MC, MKC are different index construction and instruction methods for three different index compression strategies.
test : experiment codes.
tool : tools that used in the main algorithms.
util : some config codes, like dataset path, dataset reader and so on.
